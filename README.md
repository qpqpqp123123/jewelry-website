# 珠宝展示网站

## 使用说明

### 1. 快速启动
直接双击打开 `index.html` 文件即可在浏览器中查看网站。

### 2. 修改产品信息
打开 `script.js` 文件，找到 `products` 数组，修改以下内容：
- `name`: 产品名称
- `price`: 产品价格
- `image`: 产品图片路径

### 3. 替换产品图片
将你的珠宝图片放在 `jewelry-website` 文件夹中，然后修改 `script.js` 中的图片路径。
例如：`image: "images/necklace1.jpg"`

### 4. 修改联系方式
打开 `index.html` 文件，在 footer 部分修改：
- 电话号码
- 邮箱地址
- 实际地址

### 5. 部署到服务器
将整个 `jewelry-website` 文件夹上传到你的服务器即可。

## 文件结构
```
jewelry-website/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 脚本文件
└── README.md       # 说明文档
```

- 简洁优雅的界面

- 联系方式和咨询按钮
